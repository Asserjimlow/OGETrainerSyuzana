Практическая часть. Задания по теме "Кодирование информации"

1. Перевод чисел между системами счисления:
а) Переведите число 45 из десятичной системы в двоичную.
б) Переведите число 10110(2) в десятичную систему.
в) Переведите число 2F(16) в десятичную систему.
г) Переведите число 77 из десятичной системы в шестнадцатеричную.

2. Определение объема информации:
а) Сколько информации несёт сообщение из 120 символов, если используется алфавит из 64 символов?
б) Каков объем изображения 640*480 пикселей с глубиной цвета 24 бита?
в) Какой объём будет занимать текст из 300 символов в кодировке UTF-8 (1 символ = 1 байт)?

3. Подсчет количества символов в сообщении:
а) Сообщение состоит из 5 слов, каждое по 8 символов. Сколько всего символов (если учитывать пробелы)?
б) Сколько байт потребуется для хранения слова "информация" в кодировке UTF-8?

4. Задания на работу с кодировками ASCII:
а) Какой десятичный код соответствует символу 'B' в таблице ASCII?
б) Какой символ соответствует коду 97(10)
в) Представьте слово "Hi" в виде двоичных кодов ASCII.

5. Анализ графических изображений:
а) Вычислите объем файла изображения размером 800?600 пикселей при глубине цвета 16 бит.
б) Сколько байт потребуется для хранения изображения 100?100 пикселей при глубине 8 бит?

6. Задания ОГЭ по теме (типовые задания 1–3):
а) Сколько бит потребуется для хранения сообщения, состоящего из 200 символов, если используется алфавит из 32 символов?
б) Определите объем графического файла, если известно: 640?480 пикселей, 16 бит/пиксель.
в) Найдите количество информации в сообщении, состоящем из 40 символов, если алфавит содержит 128 символов.
